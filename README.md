## ✂️ Clone Application
- ```git clone https://github.com/DaviQuental/library-project```
- ```cd library-project```

## ⚙️ Run Application with NPM
- ```npm install```
- ```npm start```

## ⚙️ Run Application with YARN
- ```yarn install```
- ```yarn start```
